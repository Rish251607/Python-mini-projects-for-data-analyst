🚀 Python-mini-projects-for-data-analyst

This repository contains a collection of Python projects focused on **Data Analysis, Data Cleaning, Automation, and API-based applications**.  
These projects demonstrate practical problem-solving and real-world analytical thinking.

---

📊 Data Analysis Projects

1. Stock Market Time-Series Analysis
Analyzed historical stock price data to identify trends, volatility, and moving averages (MA20, MA50).  
Converted raw data into time-series format and visualized long-term growth patterns.

2. Employee Salary & Performance Analysis
Performed exploratory data analysis on HR dataset using NumPy and Pandas.  
Analyzed salary distribution, performance ratings, and attrition to generate business insights.

3. Customer Churn Analysis
Identified key churn drivers using grouping and comparison techniques.  
Built rule-based risk segmentation to classify high-risk customers without using machine learning.

4. Weather Data Analysis
Analyzed time-series weather data (temperature, humidity, wind speed).  
Performed yearly and monthly analysis to identify seasonal patterns and trends.

5. Sales Data Analysis
Analyzed sales dataset to identify monthly revenue trends and top-performing product lines.  
Handled missing values using business logic and created visualizations for insights.

6. 🧹 Data Cleaning Project - Netflix Dataset Cleaning
Cleaned a real-world messy dataset by handling missing values, fixing data types, removing duplicates, and standardizing text values.  
Prepared dataset for further analysis.

---

📱 Data Processing Project

7. Instagram Data Parsing
Built a Python-based parser to convert unstructured Instagram profile data into structured format.  
Extracted key metrics like followers, posts, and categories.
